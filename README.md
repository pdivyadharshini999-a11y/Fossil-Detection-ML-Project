#  Fossil Presence Prediction using Logistic Regression

##  Overview

This project predicts whether a fossil is present at a geological site based on features such as:

* Depth
* Rock Hardness
* Soil Moisture
* Mineral Content
* Region Type

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn

---

##  Workflow

* Data loading using Pandas
* Label Encoding for categorical feature
* Train-test split
* Feature scaling using StandardScaler
* Logistic Regression model training
* Prediction and evaluation using accuracy

---

## Result

The model predicts:

* **1 → Fossil Found**
* **0 → No Fossil Found**
